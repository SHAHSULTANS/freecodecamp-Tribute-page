Key Feature Of This Projects

  • Bootstrap Integration: The page includes Bootstrap for styling, providing layout and predefined classes like `btn`, `container`, and `bg-light`.
  • Custom Styles: It contains custom styles for elements like `.question-card` to add shadows and padding, and `.btn` states for correct/incorrect answers.
  • Quiz Questions: The quiz features an array of multiple-choice questions, each with several options and one correct answer.
  • Timer for Questions: Each question has a countdown timer of 12 seconds, displayed using a `` with the ID `timer`.
  • Answer Validation: The script checks whether the selected answer is correct or incorrect, visually highlighting the answer buttons in green or red.
  • Dynamic Question Loading: Questions are loaded dynamically from an array, and users can navigate through the quiz by answering each question.
  • Score Tracking: The score is calculated based on correct answers and displayed at the end of the quiz.
  • Session Storage: The quiz state (current question and score) is saved in `sessionStorage` to preserve progress if the page is reloaded.
  • Certificate Generation: After completing the quiz, users can generate a PDF certificate using the `jsPDF` library, which includes their name and score.
  • Responsive Design: The layout is responsive, with question cards centered and the UI adapting to different screen sizes.
